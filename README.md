# Demo Trading UI

**Description:** Demo trading web app showing live crypto prices with fake money for learning purposes. Built with React, TypeScript, and Tailwind CSS. Perfect for practicing UI design, state management, and trading logic without any real financial risk.

## Features

- Real-time crypto prices (BTC, ETH, etc.)  
- Candlestick charts and price updates  
- Simulated buy/sell orders with fake balance  
- Trade history and PnL calculation  
- Dark theme dashboard layout  
- Fully responsive for desktop and mobile  

## Tech Stack

- React + Vite  
- TypeScript  
- Tailwind CSS  
- Zustand (state management)  
- Fetch / WebSocket for live price feeds
